## 功能特点
- 纯Win32编程，最大程度保留兼容性，**甚至连XP都支持**（个别函数重编译）
- 大量方便的窗口操作，操作简单
- 支持**子窗口嵌套**操作，满足你的特殊需求
- 添加了**枚举**功能，就没有我找不到的窗口
- 支持对窗口**发送消息**，Win32开发者狂喜
- 支持**UIACCESS级置顶**，方便操作（需管理员权限）
